# Import necessary libraries
import pandas as pd
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

# Load the dataset
data = pd.read_csv('homeprices.csv')

# Define the features (X) and target (y)
X = data[['area']]
y = data['price']

# Create a Linear Regression model and fit it to the data
model = LinearRegression()
model.fit(X, y)

# Predict prices
predicted_prices = model.predict(X)

# Plot the results
plt.scatter(X, y, color='blue', label='Actual Prices')
plt.plot(X, predicted_prices, color='red', label='Predicted Prices')
plt.xlabel('Area (sq ft)')
plt.ylabel('Price ($)')
plt.legend()
plt.show()

# Display the model's coefficient and intercept
print(f"Coefficient: {model.coef_[0]}")
print(f"Intercept: {model.intercept_}")

Heres a breakdown of what each part of the code does and why its important:

1. **Import Libraries**  
   ```python
   import pandas as pd
   from sklearn.linear_model import LinearRegression
   import matplotlib.pyplot as plt
   ```
   - `pandas` is used to handle and manipulate data.
   - `LinearRegression` from `scikit-learn` is the linear regression model well use for predicting prices.
   - `matplotlib.pyplot` allows us to plot and visualize the results.

2. **Load the Dataset**  
   ```python
   data = pd.read_csv('homeprices.csv')
   ```
   - This reads in the `homeprices.csv` file and stores it in a DataFrame called `data`. Each row in this file represents a house, with columns for the area (`area`) and price (`price`).

3. **Define Features (X) and Target (y)**  
   ```python
   X = data[['area']]
   y = data['price']
   ```
   - `X` is the feature, which is the `area` column in this case. Wrapping `data[['area']]` in double square brackets ensures that `X` will remain a DataFrame with a single column.
   - `y` is the target variable or the label we want to predict, so we set it to `data['price']`.

4. **Create and Train the Linear Regression Model**  
   ```python
   model = LinearRegression()
   model.fit(X, y)
   ```
   - We first create a `LinearRegression` model object called `model`.
   - `.fit(X, y)` trains the model by finding the best-fit line for the data, which minimizes the difference between the actual and predicted values.

5. **Predict Prices Using the Model**  
   ```python
   predicted_prices = model.predict(X)
   ```
   - `.predict(X)` uses the trained model to predict prices based on `X` (areas in this case).
   - The predictions are stored in `predicted_prices`, which can be compared to the actual prices in `y`.

6. **Plot the Results**  
   ```python
   plt.scatter(X, y, color='blue', label='Actual Prices')
   plt.plot(X, predicted_prices, color='red', label='Predicted Prices')
   plt.xlabel('Area (sq ft)')
   plt.ylabel('Price ($)')
   plt.legend()
   plt.show()
   ```
   - `plt.scatter(X, y, color='blue', label='Actual Prices')` plots the actual data points (area vs. price) as blue dots.
   - `plt.plot(X, predicted_prices, color='red', label='Predicted Prices')` draws the linear regression line representing the models predictions.
   - `plt.xlabel()` and `plt.ylabel()` label the axes.
   - `plt.legend()` and `plt.show()` add a legend and display the plot, showing the fit line and how well it represents the data.

7. **Display the Models Coefficient and Intercept**  
   ```python
   print(f"Coefficient: {model.coef_[0]}")
   print(f"Intercept: {model.intercept_}")
   ```
   - `model.coef_` gives the slope of the line, representing how much the price changes with each unit of area.
   - `model.intercept_` is the point where the line crosses the price axis, representing the predicted price for a house of zero area.
   - These values form the linear equation for prediction: **`price = (coefficient * area) + intercept`**.

### Summary of the Workflow:
1. **Prepare data**: Separate the features (X) and target (y).
2. **Train model**: Fit the linear regression model to the data.
3. **Predict**: Use the trained model to predict prices.
4. **Visualize**: Plot actual vs. predicted prices to analyze the fit.
5. **Interpret**: Display model parameters to understand the relationship between area and price.
